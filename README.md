# Python Calculator App

A simple desktop calculator application built with Python and Tkinter.

## Requirements
- Python 3.7 or newer
- Tkinter (built in on most Python installs)

## Run
```bash
python calculator.py
```

## Features
- Basic arithmetic: `+`, `-`, `*`, `/`
- Decimal support
- Clear `C`
- Backspace `←`
- Toggle sign `±`
- Simple UI with buttons and display
