# StockAnalyzer

A basic AI-driven stock analysis application (MVP).

## Features

- Fetch historical stock data
- Basic financial analysis (returns, moving averages)
- Simple sentiment analysis
- ARIMA-based price prediction
- Data visualization

## Installation

1. Install dependencies: `pip install -r requirements.txt`
2. Run the script: `python stock_analyzer.py`

## Usage

Modify the ticker in the script (e.g., 'AAPL') and run.

This is a starting point for the full enterprise application.
